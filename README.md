# ☁️ CloudPulse – Modular JavaScript Weather App


## 📂 Folder Structure
```
├── src
│ ├── assets # Styles and Images Folder
│ │ ├── images # Images and SVGs files
│ │ └── styles # CSS files
│ ├── modules/ # Your modular JS files
│ ├── index.js # Main entry point
│ └── template.html # Base HTML template
├── dist # Auto-generated output
├── package-lock.json
├── package.json # project dependencies and scripts
├── README.md
├── webpack.common.js # webpack common config for dev and prod
├── webpack.dev.js # webpack config for development
└── webpack.prod.js # webpack config for production
```
